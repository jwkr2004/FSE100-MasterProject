What is Human Improvement?

Human Improvement is an application that is designed for recovering stroke patients trying to bring their cognitive abilities back to normal. However, anyone can use this app, either for fun or to improve skills. Human Improvement currently has three games/activies. The Typing Therapy game focuses the the user's ability to read words on the screen, type that word out correctly, all while trying to be as quick as possible. The Memory Matcher game focuses on the user's short term memory skills. The Big Iron game focuses on the user's hand eye coordination as the user must look at the screen to find the location of the target and then click it with the mouse. All of the games are very customizable and this application is compatable with mobile devices.
